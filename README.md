# Singapore Explorer: The Changi Challenge

A premium interactive journey through Singapore, beginning at the world's best airport.

## Overview

This is not a quiz game with a travel theme. It is a premium interactive travel experience where learning about Singapore happens naturally through exploration, conversation, observation, and carefully designed challenges.

## Vision

Imagine if Singapore Tourism Board and Changi Airport Group built a game together—not to advertise, but to make people fall in love with Singapore.

## Features (Version 1)

- **Changi Airport Experience**: Explore the world's best airport through 5 interactive missions
- **Hawker Centre Journey**: Discover authentic local food culture and community
- **Passport System**: Collect stamps as you complete destinations
- **Premium Design**: Glassmorphism UI, elegant animations, thoughtful interactions
- **Mobile Responsive**: Designed for all devices
- **Local Storage**: Save your progress

## Getting Started

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Start your journey through Singapore

## Design Philosophy

- **Premium Experience**: Inspired by Changi Airport, Singapore Airlines, Apple, National Gallery
- **Authentic Singapore**: Represents multicultural, modern, heritage Singapore
- **Respectful Learning**: Never over-explains, rewards discovery
- **Emotional Journey**: Each destination teaches through feeling, not just facts

## Experience Principles

1. **Always Travelling** - Every screen feels like another step of a journey
2. **Changi is the Gateway** - Airport welcomes, orients, and serves as hub
3. **Every Destination Teaches Differently** - Unique learning method per location
4. **Curiosity Before Trivia** - Show, then ask
5. **Reward Discovery** - Passport stamps, souvenirs, achievements
6. **Authentic Singapore** - Chope, Kopi, HDB, hawkers, multiculturalism
7. **Premium Design** - No cheap animations, gamey UI, or flashing buttons
8. **Respect Player Intelligence** - Let players infer, discover, observe
