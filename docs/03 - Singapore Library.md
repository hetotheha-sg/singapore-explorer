# 🇸🇬 Singapore Library

## Purpose

The Singapore Library is the creative palette of the project.

It contains authentic cultural references, atmosphere, sounds, memories and ideas that make the experience unmistakably Singapore.

Nothing here is mandatory.

Everything here exists to inspire.
